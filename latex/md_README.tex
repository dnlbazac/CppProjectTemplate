This is a template for C++ projects. What you get\+:


\begin{DoxyItemize}
\item Library, executable and test code separated in distinct folders
\item Use of modern CMake for building and compiling
\item External libraries installed and managed by
\begin{DoxyItemize}
\item \href{https://github.com/cpm-cmake/CPM.cmake}{\texttt{ CPM}} Package Manager OR
\item \href{https://conan.io/}{\texttt{ Conan}} Package Manager OR
\item \href{https://github.com/microsoft/vcpkg}{\texttt{ VCPKG}} Package Manager
\end{DoxyItemize}
\item Unit testing using \href{https://github.com/catchorg/Catch2}{\texttt{ Catch2}} v2
\item General purpose libraries\+: \href{https://github.com/nlohmann/json}{\texttt{ JSON}}, \href{https://github.com/gabime/spdlog}{\texttt{ spdlog}}, \href{https://github.com/jarro2783/cxxopts}{\texttt{ cxxopts}} and \href{https://github.com/fmtlib/fmt}{\texttt{ fmt}}
\item Continuous integration testing with Github Actions and \href{https://pre-commit.com/}{\texttt{ pre-\/commit}}
\item Code coverage reports, including automatic upload to \href{https://codecov.io}{\texttt{ Codecov}}
\item Code documentation with \href{https://doxygen.nl/}{\texttt{ Doxygen}} and \href{https://franneck94.github.io/CppProjectTemplate/}{\texttt{ Github Pages}}
\item Tooling\+: Clang-\/\+Format, Cmake-\/\+Format, Clang-\/tidy, Sanitizers
\end{DoxyItemize}\hypertarget{md_README_autotoc_md1}{}\doxysection{Structure}\label{md_README_autotoc_md1}

Library code goes into \href{src/}{\texttt{ src/}}, main program code in \mbox{[}app/\mbox{]}(app) and tests go in \href{tests/}{\texttt{ tests/}}.\hypertarget{md_README_autotoc_md2}{}\doxysection{Software Requirements}\label{md_README_autotoc_md2}

\item CMake 3.\+21+

\item Doxygen

\item MSVC 2017 (or higher), G++9 (or higher), Clang++9 (or higher)
